Dispatch · What it is

Apple Podcasts holds no audio. It’s a directory — every show points at an RSS feed where each episode carries a direct link to its file. podget walks that chain and hands you the audio.

Give it a show, pick episodes from an interactive list, and they land on your disk — named, dated, resumable. One small command-line tool. No subscription, no account, nothing to trust.

Three moves, start to finish
01

Search

Find any show by name. iTunes’ open catalog — 4 million+ feeds, no key required.

$ podget search "睡前故事"
02

Pick

Hand it a show and choose from an interactive list — arrow keys, space to toggle, one or many.

$ podget get 1532755821
03

Download

Resumable, with a live progress bar. Saved as YYYY-MM-DD - title.mp3.

$ podget get 1532755821 --latest 3

How it works

No scraping · No private API · Just the feeds publishers already hand out

What’s in the box

Arrow-key, multi-select checklist. Grab one episode or twenty.

Live progress

Spinners for fetches; size · speed · ETA bars for downloads.

✓

Apple · RSS · 小宇宙

Shows, raw feeds, or a pasted episode link — all resolve to the file.

Resumable
